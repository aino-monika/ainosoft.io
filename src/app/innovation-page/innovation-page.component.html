AppOps is a cloud application infrastructure framework that uses cloud neutral api’s to develop, deploy and run microservices with a web and mobile application to use them.

Appops improves the interoperability between different programming ecosystems and helps the modules in an app ‘Talk’ better. Learn more



Artikulate is an AI driven article extraction platform deployed to fetch, convert, sort and display newspaper articles from various online sources in a well-formatted, tabulated and searchable format.

Using machine learning algorithms trained on various online print formats, artikulate can extract articles with 99.99 percent accuracy at the rate of a million pages per hour with zero downtime coupled with state-of-the-art storage and retrieval mechanisms.Learn more



xTractor is an easy to use and configurable scraping solution using xPath. It is able to bypass most blocking and throttle implementations with innovative approaches and analyze and convert unstructured html to structured xml. It is made for high performance and can be used at scale.
